CSS problems? Houston, we have a solution.

The No-Nonsense, Batteries-Included CSS Framework

Designed with simplicity in mind

SpaceBox is a collection of CSS classes designed to make your life easier. If you have a lightbulb moment and want to get the most out of the fleeting moment of inspiration, SpaceBox can help you put together an effective layout quickly, saving you from spending hours tinkering with the CSS to make everything look nice.

Fast prototyping?

Fastest prototyping in the west, kiddo

I come up with a lot of random ideas, and I like to throw together quick prototypes while the idea is fresh. I always hate when I'm being held back by clunky CSS (or my own obsession with making my projects look nice) when I could just be writing code and having fun implementing functionality. That's why I made SpaceBox, which I designed with this exact type of scenario in mind.

Using SpaceBox, you can throw together a nice looking layout in just minutes, and the code will be clean and super easy to read and expand upon later. SpaceBox makes designing layouts fast and effortless.
